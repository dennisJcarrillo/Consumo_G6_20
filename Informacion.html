Vuelo Pasajero Reserva

Este formulario sirve para consultar a la base de datos en Postgres
sobre los vuelos que hay actualmente con la función CargarVuelos.
La función CargarVuelo sirve para mandar los datos de x vuelo al
apartado del formulario, de acuerdo a su código
La función AgregarVuelo nos ayuda a crear un nuevo vuelo llenando
los datos necesario y así ingresarlo a la base de datos.
ActualizarVuelo es una función que sirve para llamar a un vuelo existente de la
base de datos, y así poder actualizar sus datos, exceptuando obviamente el código de vuelo.
Por último la funcion BorrarVuelo sirve para, como su nombre lo
dice, eliminar un vuelo de la base de datos de acuerdo a su código.

Este formulario sirve para poder consumir las API´S Pasajero la cual nos provee datos que obtiene desde una base de datos en PostgreSQL.
Dicho formulario cuenta con una parte que muestra de forma inmediata los registros y todos sus campos que existen en la tabla Pasajero de la base de datos,
mencionar también al final de cada registro tenemos dos botones,
uno para editar un registro y otro para eliminar.
Además cuenta con otro apartado o formulario para poder insertar o modificar un registro.
